# Cost Assumptions & Future Improvements

## Current Model (v1 - Hackathon)
- Install cost: $500/m² flat rate (simplified)
- No fixed costs per location
- No land cost (Green P = city owned = $0)

## Planned Improvement (v2 - Post Hackathon)
Break down installation cost into real components:

| Component | Cost/m² | Notes |
|---|---|---|
| Steel canopy structure | $150-200/m² | Verify with steel contractor |
| Solar panels | $150-200/m² | Commercial grade 400W |
| Electrical/inverters | $50-80/m² | DC to AC conversion |
| Labor | $30-50/m² | Installation workers |
| **Total variable** | **~$455/m²** | |

| Fixed Cost per Location | Estimate | Notes |
|---|---|---|
| Grid connection | $20,000-50,000 | Alectra connection fee |
| Permits & engineering | $15,000-30,000 | City of Toronto |
| **Total fixed** | **~$40,000** | |

## Real Cost Formula (v2)

## Land Cost
Green P lots are owned by City of Toronto.
No land acquisition cost needed.
This is a major advantage over private rooftop installations.

## Why Green P is cost-effective vs private rooftops
1. No land cost
2. No negotiation with private owners
3. City can fast-track permits on own property
4. Revenue goes back to city budget
5. Scalable — 120+ locations across Toronto
